Esterel

Esterel is a synchronous programming language for describing reactive systems. These pages include

  • A simple backhoe loader example and simulator, for use as a tutorial introduction to Esterel,
  • Some notes on building programs compiled with the v5_92 tools on Win32,
  • and similarly for fixing "__ctype_b" problems on Linux, and,
  • An a2ps stylesheet for pretty-printing Esterel v5 programs.
